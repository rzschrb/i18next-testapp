# i18next Test App

App to understand on how to use i18next lib using React and Vite.

Might be or not updated.

## Requirements

 - Node.js
 - NPM

## Run Locally

Clone the project

```bash
git clone https://github.com/rzschrb/i18next-testapp.git
```

Go to the project directory

```bash
cd i18next-testapp
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
